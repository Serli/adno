Création d'un projet

Rendez-vous sur la page d'accueil d'ADNO, puis, renseignez le fichier info.json de votre oeuvre au iiif puis cliquez sur le bouton "Créer un nouveau projet"

Une fois avoir cliqué sur la bouton vous allez être redirigé vers la page de création du projet, sur cette page il vous sera demandé de renseigner un titre pour votre projet (obligatoire) ainsi que description (facultative).

Après avoir rempli ces informations pour votre projet il vous suffira de cliquer sur le bouton bleu "créer mon projet"

Une fois votre projet créé, vous allez être redirigé vers le mode prévisualisation de celui-ci.
